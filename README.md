# DevOps Practice Repository

This repository is a personal learning project focused on DevOps tools and workflows.  
It contains exercises, practice files, and configurations for Docker, InfluxDB, Grafana, Jenkins, GitHub Actions, and end-to-end automation.

## Structure

- **git-basico/** – Practice files for Git basics, commits, branching, and learning Git workflows.  
- **docker-influx-grafana/** – Docker containers setup, InfluxDB, and Grafana practice, monitoring examples.  
- **jenkins-pipelines/** – Jenkins pipelines, scripts, and CI/CD examples.  
- **github-actions/** – GitHub Actions workflows and automation examples.  

## Purpose

- Track learning progress and experiments with DevOps tools.  
- Practice end-to-end automation and CI/CD pipelines.  
- Maintain a historical record of commands, scripts, and configurations.  

## How to use

1. Explore each folder to see daily or topic-specific exercises.  
2. Check the `.md` files for explanations, commands, and notes.  
3. Pull or clone the repository to try out the examples locally in WSL/Linux.  

> Every commit in this repository reflects a learning milestone or a mini-project.
